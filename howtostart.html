How Do I Start?

Setting up the Raspberry Pi is simple and can be done in a few simple steps:

  1. First, you must go to the Raspberry Pi website and install NOOBS opertating software on a micro SD card
  2. Next, insert the micro SD card into the pi and connect a keyboard and mouse using the USB ports
  3. Then, connect a HDMI monitor or TV to the Raspberry Pi using and HDMI cable
  4. Finally, plug the micro-USB cable to the Raspberry Pi and then into a power outlet

Once all the connections are made, turn on the tv. A menu will load up, choose the operating system you wish to use
and it will start downloading. And just like that... the Pi is set up!
